什么是gnt?

A GNT file stores multiple grey-scale character images sequntially. Each image has a record for total number of bytes, the class table(2-byte GB code), the width and height, and the bitmap(on byte per pixel). The gray-scale image has background pixels uniformly labeled as 255, and so, can be easily be converted to binary image. 来自于书：《Advances In Chinese Document And Text Processing edited by Liu Cheng-lin, Lu Yue》
